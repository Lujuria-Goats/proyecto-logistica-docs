# Documentación del Proyecto Logística

Este repositorio contiene la documentación técnica y funcional del sistema de logística, estructurada para su publicación y navegación en Wiki.js.

## Estructura del Repositorio

- **01-ARQUITECTURA**: Visión general, diagramas y decisiones técnicas.
- **02-SERVICIOS**: Documentación detallada de microservicios (Backend C#, Java AI, Frontend).
- **03-DEVOPS-CI-CD**: Guías de despliegue, pipelines y configuración de servidores.
- **04-MANUAL-USUARIO**: Guías para el usuario final y resolución de problemas.

## Contribución

1.  Para editar, navega a la sección correspondiente.
2.  Usa formato Markdown estándar.
3.  Mantén el prefijo numérico en los archivos para conservar el orden.
